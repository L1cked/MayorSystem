# Menus

Most server-owner setup can be done from in-game menus instead of editing files directly. Use `/mayor` for player flows and `/mayor admin` for election setup, perk management, display settings, health checks, reloads, and maintenance tools.

## Player Menus
- Main menu
- Status menu
- Election ranking menu
- Vote menu and vote confirm menu
- Candidate menu
- Candidate perk catalog, section, and selected-perks menus
- Candidate custom perk request menu
- Apply menus
- Step-down confirm menus
- Mayor profile menu

## Admin Menus
- Admin home
- Maintenance and reset tools
- Monitoring, audit, and health
- Candidate moderation and apply-ban tools
- Perk catalog, refresh, and request tools
- Election, fake vote, and force-elect tools
- Settings menus
- Display reward target menus
- Messaging menus
- NPC and hologram display menus

Menu text and layout are configured in `gui.yml`.

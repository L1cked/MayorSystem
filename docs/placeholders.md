# PlaceholderAPI

If PlaceholderAPI is installed, MayorSystem registers these placeholders:

```text
%mayorsystem_title_name%
%mayorsystem_title_name_lower%
%mayorsystem_title_display%
%mayorsystem_title_tag%
%mayorsystem_leaderboard_term%
%mayorsystem_leaderboard_<pos>_name%
%mayorsystem_leaderboard_<pos>_display_name%
%mayorsystem_leaderboard_<pos>_votes%
%mayorsystem_leaderboard_<pos>_uuid%
```

`<pos>` starts at `1`. Empty slots return an empty string.
